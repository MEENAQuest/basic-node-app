To execute the application, please follow the following instructions - 

1. Open VS Code, and click on 'File' menu, then 'Open Folder'.
 
2. Select the 'Basic Node App' folder. This will open up the Basic Node application on VS Code. 

3. In VS Code, click on the 'Terminal' menu item and then 'New Terminal'. This will open up the terminal at the bottom of VS Code. 

4. Execute the command 'npm install' on the terminal. This will install all the required dependencies. 

5. Next, execute the command 'node index' to run your application. 

6. Finally, open your browser and on the address bar, type 'localhost:3000' to view your node project.
